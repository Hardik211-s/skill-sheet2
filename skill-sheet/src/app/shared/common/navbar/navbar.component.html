📌 How-To Guides

  • How to Add a Skill?
    Go to the Skills section → Click Add Skill → Fill details → Click Save.
  • How to Generate a Skill Report?
    Navigate to Reports → Select filters → Click Generate Report → Download file.
  • How to Update User Information?
    Go to Profile Settings → Edit details → Click Save Changes.
